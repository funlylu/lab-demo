二、Firefox浏览器

1、支持Firefox所有版本@-moz-document url-prefix() { .selector { property: value; } }

 在网上还看到一些hack方法，经验证都不可行。下面这些设置在其它浏览器照样能被识别，作者测试了chrome和ie以及Firefox自身都能识别。
 例如：
  1、支持geoko内核的浏览器  *>.selector { property: value; }
  2、#selector[id=selector] { property: value; }
实例测试
这个实例测试了上面说道的三个方法。读者可以用调试工具自行调试查看即可

三、webkit内核

@media screen and (-webkit-min-device-pixel-ratio:0) { Selector { property: value; } }
实例测试
在webkit内核浏览器中背景色显示为橙色
